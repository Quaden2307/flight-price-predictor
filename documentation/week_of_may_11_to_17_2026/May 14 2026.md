Collector run 7 came in clean — 0 failures, 4043 offers, back to the ~7 minute baseline after two rough days. The retry-with-backoff code from yesterday never got exercised since the network was healthy, so it's in place but still untested in the wild. Cumulative dataset now 28,428 rows across 7 runs. Logged the bounce-back in live_data_logging.md.

Spent the rest of the day on EDA in 03_live_data_visualization.ipynb. Three plots so far: price histogram, price vs lead_time scatter, and side-by-side histograms of both variables. The big finding is that the price distribution is clearly bimodal — a primary peak around $300 (domestic short-haul) and a secondary hump around $800-900 (long-haul international). The scatterplot looked confusing at first because the dense band has no obvious slope, but that's because the bimodality is dominating the spread and washing out any within-route lead-time signal. Realized I need is_international and distance_km features before I can see anything useful in this data.

Also pulled the type column thinking it might be a free hub_tier starter; turned out useless since every airport is classified as "large_airport." Left the column in but it provides no signal.
